Recharts 使用 inline SVG style，不识别 Tailwind className。颜色必须用：
```typescript
// 正确：CSS 变量字符串
stroke="var(--sy-accent)"    // NOT className="stroke-sy-accent"
fill="var(--sy-accent)"
// 参考 page.tsx 中已有的 priorityColors 写法
```

```typescript
interface TrendDataPoint {
  iteration_name: string;
  testcase_count: number;
  p0_count: number;
  coverage_rate: number;  // 0-100
}
```

SourcePieChart 数据结构（来自 /dashboard/quality-stats by_source）：
```typescript
interface SourceData {
  ai_generated: number;
  imported: number;
  manual: number;
}
// 转换为 PieChart data：
const pieData = [
  { name: 'AI 生成', value: data.ai_generated, color: 'var(--sy-accent)' },
  { name: '历史导入', value: data.imported, color: 'var(--sy-info)' },
  { name: '手动创建', value: data.manual, color: 'var(--sy-purple)' },
]
```

Recharts PieChart 环形图模式：
```typescript
<PieChart width={220} height={220}>
  <Pie data={pieData} innerRadius={55} outerRadius={85} dataKey="value" paddingAngle={2}>
    {pieData.map((entry, i) => <Cell key={i} fill={entry.color} />)}
  </Pie>
  <Tooltip formatter={(value) => [`${value} 条`, '']} />
  <Legend />
</PieChart>
```

Recharts LineChart 模式：
```typescript
<LineChart data={trendData} margin={{top:5,right:20,left:0,bottom:5}}>
  <CartesianGrid strokeDasharray="3 3" stroke="var(--sy-border)" />
  <XAxis dataKey="iteration_name" tick={{fill:'var(--sy-text-2)',fontSize:11}} />
  <YAxis tick={{fill:'var(--sy-text-2)',fontSize:11}} />
  <Tooltip contentStyle={{background:'var(--sy-bg-1)',border:'1px solid var(--sy-border)'}} />
  <Legend />
  <Line dataKey="testcase_count" name="用例总量" stroke="var(--sy-accent)" dot={false} strokeWidth={2} />
  <Line dataKey="p0_count" name="P0 数量" stroke="var(--sy-danger)" dot={false} strokeWidth={2} />
  <Line dataKey="coverage_rate" name="覆盖率%" stroke="var(--sy-info)" dot={false} strokeWidth={2} />
</LineChart>
```
